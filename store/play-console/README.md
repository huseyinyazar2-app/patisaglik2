# Pet Help Play Console Assets

Bu klasor Play Console'a yuklenecek magazaya ozel gorselleri tutar.

- `pet-help-play-icon-512.png`
  - Play Console app icon
  - 512 x 512 PNG
  - Opak, tam kare artwork; Play Console maskelemesini kendi uygular.
  - Sadece store listing icindir; Android launcher ikonlari ayrica `android/app/src/main/res/` altindadir.

- `pet-help-feature-graphic-1024x500.jpg`
  - Play Console feature graphic
  - 1024 x 500 JPG
  - Varsayilan TR magazasi icin hazirlandi.

Notlar:
- Android uygulama ici launcher/adaptive ikonlar daha once `android/app/src/main/res/mipmap-*` ve `drawable` kaynaklarina islendi.
- Screenshot setleri cihaz/emulator uzerinde gercek mobil ekranlar test edildikten sonra alinmali.
- App adi: `Pet Help`
- Paket adi: `app.pethelp.mobile`
